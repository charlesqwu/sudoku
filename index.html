Classic Sudoku is a number placing puzzle, based on a 9x9 grid with several given numbers. The object is to place the numbers 1 to 9 in the empty squares so that each row, each column, and each 3x3 block contains the same number only once. (For more information, see: https://en.wikipedia.org/wiki/Sudoku)

Sudoku Rules:

  1. Number can appear only once in each row with all numbers in each row summing up to 15;
  2. Number can appear only once in each column with all numbers in each column summing up to 15;
  3. Number can appear only once in each block with all numbers in each block summing up to 15;

What Have Been Accomplished
  1. The game board's UI was taken from https://codepen.io/cristiancanea/pen/GhLpI, but I have styled using the Bootstap framework;
  2. Now the game is hooked with a backend -- each game action will make an Ajax call to save the playing step into database;
  3. Clicking on the "Review Played Games" will show such saved steps. (Currently, there is no user information; and therefore all players' data will be displayed.)
Possible Future Enhancements
  1. A function to show or not show playing steps concurrently;
  2. Replay a game: Each already-played game will be showed with a "Replay" button -- clicking on it will replay the game on the left-side with a scaled-down time frame;
  3. A "Demo" button to show a dynamic "tutorial" -- that is, playing a demo game with explanation and tips.
